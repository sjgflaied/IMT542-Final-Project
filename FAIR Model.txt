Mental Health Crisis Early-Warning System (MHCEWS): The MHCEWS generates ZIP-code-level weekly risk records, each uniquely identified by a composite key of zip_code + ISO 8601 week_start_date and described through a four-layer metadata structure (Identity, Signal, Prediction, Action), providing both human-readable labels and machine-readable typed values so that downstream systems can interpret the data without additional context (‘F’). Records are accessible via a REST API using standard HTTP protocols that are open, free, and universally implementable, with OAuth 2.0 authentication for sensitive data streams (e.g., real-time dispatch logs) while aggregate risk outputs remain openly available (‘A’). 

All data fields follow snake_case conventions aligned with JSON-LD practices, and geographic identifiers use standardized vocabularies such as Census FIPS and ZCTA ZIP codes maintained by the U.S. Census Bureau, ensuring interoperability across systems that adopt these widely used standards (‘I’). Each record includes a detailed provenance block capturing data_source, model_version, and data_freshness timestamps for every contributing input, enabling full traceability of how risk scores are generated (‘R’). Public outputs are released under a Creative Commons Attribution 4.0 license, ensuring clarity in reuse conditions and supporting transparency and reproducibility (‘R’).
